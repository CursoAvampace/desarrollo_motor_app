## Descripción
Esta aplicación web está diseñada para presentar, de forma interactiva, información fundamental sobre el desarrollo motor y cognitivo en niños. Se muestra un resumen que abarca etapas del desarrollo, trastornos motrices y estrategias de intervención.

## Características
- Resumen interactivo y visual de información didáctica.
- Diseño moderno, limpio y adaptable a distintos dispositivos (responsive).
- Arquitectura modular que facilita mantenimiento y escalabilidad.

## Tecnologías Utilizadas
- **Backend:** Python 3.9+, FastAPI, Pydantic, Uvicorn.
- **Frontend:** React, JavaScript (ES6+), Webpack, Babel, Axios, Bootstrap.
- **Otras Herramientas:** Node.js, npm o yarn.

## Requisitos
1. Python 3.9+ (descargable desde [python.org](https://www.python.org/downloads/)).
2. Node.js y npm (o yarn) (descargable desde [nodejs.org](https://nodejs.org/)).
3. Un editor de código (por ejemplo, Visual Studio Code).
